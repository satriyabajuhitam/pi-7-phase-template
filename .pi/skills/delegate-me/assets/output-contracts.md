# Output Contracts

Use these parent-facing output shapes by default.

## No-spawn verdict

Return exactly these sections and no others:
- Delegation verdict
- Policy status
- Why not
- Smallest next step

## Generic spawn verdict

Return exactly these sections and no others:
- Delegation verdict
- Policy status
- Chosen mode
- Objective
- Source artifacts
- Spawn result summary
- Risks
- Recommended next step

## Phase 2 — Research

Return exactly these sections and no others:
- Delegation verdict
- Policy status
- Chosen mode
- Objective
- Confirmed findings
- Open questions
- Recommended next step

## Phase 4 / 5 readiness review

Return exactly these sections and no others:
- Delegation verdict
- Policy status
- Chosen mode
- Objective
- Evidence
- Readiness verdict
- Recommended next step

## Phase 6 — Execution prep

Return exactly these sections and no others:
- Delegation verdict
- Policy status
- Chosen mode
- Ticket
- Objective
- Dependencies
- Likely files
- Validation targets
- Recommended next step

## Phase 7 — QA support

Return exactly these sections and no others:
- Delegation verdict
- Policy status
- Chosen mode
- Missing scenarios
- Edge cases
- Risks
- Recommended next step
